(static) responsiveOrientation(data, $endActionCallback) → {Void}

Provides opportunity to commit actions depend on different orientations of screen. Helps to build responsivity.

Parameters:
Name Type Default Description
data Map

Map which consists of {width: Num, height: Num} => actionCallback. OR 0 => actionCallback. {width: Num, height: Num} - describes orientation, actionCallback - action that should be performed in case of reaching the orientation. 0 - if no particular orientations were reached, in that case it's possible to commit some action.
$endActionCallback Callback | Boolean false

Action at the end of adjusting element.

Example

You can call it like this:

Main.responsiveOrientation(new Map([
  [{width: 1000, height: 1400}, () => {
      doSmth();
  }],
  [{width: 1000, height: 750}, () => {
      doSmthAnother();
  }],
  [0, () => {
      okDoSmthELSE();
  }]
]));
